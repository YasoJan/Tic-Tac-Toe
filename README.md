#Tic-Tac-Toe

Tic-Tac-Toe is a "The Odin Project" assignment that involves getting a working game in the console using IIFEE factories rather than global coding. The only global code is the call to main() the entrypoint function. 

## What I learned

I learned how to encapsulate code and create private variables that cannot be modified outside. I also learned how to properly structure a program in JS from start to finish. 

All of this was created without the use of AI.
